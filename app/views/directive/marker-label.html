The marker-label directive is used to create a marker label on an existing map. It is entirely dependent on being contained within the marker directive. This is because each label is directly associated with each marker drawn within the marker directive. Therefore a label is intended to be rendered relative to each marker rendered by the marker directive.

Usage

Parameters

Required properties are in red.

Param Type Details
content expression Text to be assigned as the content of a label. Where the text is a property on a marker object.
anchor expression String expression which is evaluated at the dom level. The string should contain the x and y point position of the label relative to its current position.
class expression String expression which is evaluated at the dom level that represents the css class for this labels dom element.
style expression String expression which is evaluated at the dom level that represents the css styling for this labels dom element.
