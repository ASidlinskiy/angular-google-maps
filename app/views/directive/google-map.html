The google-map directive is the core of all other directives. It is used to instantiate a Google Map object in your page. Every other directive must be contained within a google-map for them to work.

Usage

Parameters

Required properties are in red.

Param Type Details
center expression Expression to evaluate as an object representing the map center. May be in the form of an object containing latitude and longitude properties, or New in v1.1 a GeoJSON Point object.
pan string or boolean This attribute is a flag indicate how the map's centering should behave. This tells the map to use map.panTo instead of map.setCenter. (Animate to center via pan or not.)

Note: Evaluated on the attribute only. This means it is is not watched or followed on the scope.

The pan attribute will evaluate to true if any of the following are met:

  • angular.isDefined(val)
  • val isnt null
  • val is true
  • val is "1"
  • val is "y"
  • val is "true"
control Object {} otherwise undefined / null

If some object is passed (via scope binding) functionality is appended to that object. This allows you to get "control" of the map.

The following functions are added:

getGMap:() - returns the direct reference of the google map instance being used by the directive.

refresh:(optionalCoords({latitude:, longitude}) - refreshes the current map on its current coords if no coords are specified.

example

$scope.map.control.refresh({latitude: 32.779680, longitude: -79.935493});
zoom expression Expression to evaluate as the map's zoom level. This must be a number between 1[1] and 20[2].
draggable string true if map must be draggable, false otherwise
refresh expression Expression which, if it evaluates to true, will trigger a map refresh. Useful when the map is initially hidden.
options expression Additional options to pass to the google.maps.Map constructor. See all map options
events expression

Custom events to apply to the map. This is an associative array, where keys are event names and values are handler functions. See Map Events documentation

The handler function takes three parameters:
  1. maps: the GoogleMap object
  2. eventName: the name of the event
  3. arguments: the arguments provided by Google Maps for this event type. See the Google Map Event docs
bounds expression Fit the map in the specified bounds. The expression must resolve to an object having both northeast and southwest properties. Each of those properties must have a latitude and a longitude properties.
