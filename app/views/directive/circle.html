The circle directive is used to draw circle shape layers on the map. See Circle API.

Usage

Parameters

Required properties are in red.

Param Type Details
center expression

The coordinates defining the center of the circle.

May be in the form of an object containing latitude and longitude properties, or New in v1.1 a GeoJSON Point object.
radius expression The radius of the circle
stroke expression Object to configure the line stroke. Valid properties are color (CSS hex notation), weight (int) and opacity (between 0.0 and 1.0)
fill expression Object to configure the circle fill type. Valid properties are color (CSS hex notation) and opacity (between 0.0 and 1.0)
clickable expression boolean property to tell the circle to respond to click events
draggable expression boolean property to make the circle draggable across the map
editable expression boolean property to make the circle editable by dragging control points
visible expression boolean property which shows or hides the circle
