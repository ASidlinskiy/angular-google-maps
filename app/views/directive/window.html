The window directive is used to display a single InfoWindow object on your map.

Usage

Parameters

Required properties are in red.

Param Type Details
coords expression

Expression to evaluate as an object representing the window location. May be in the form of an object containing latitude and longitude properties, or New in v1.1 a GeoJSON Point object.

Not required if the window is included within a marker. It will then use the marker's coordinates to position itself.
show expression boolean value used to show or hide the window
templateUrl expression URL to a template which will be used as the window's content
templateParameter expression object to expose to the window's template scope if templateUrl is used, or to the scope provided to the inner HTML of the directive
isIconVisibleOnClick expression boolean value indicating that the marker's icon will be shown or hidden when opening a window which is included within a marker
closeClick expression expression to evaluate when closing the window via a click event
options expression object containing InfoWindowOptions to apply to the window
